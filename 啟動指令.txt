  第一步：在 GitHub 網站上建立新倉庫


   1. 前往 GitHub (https://github.com/)。
   2. 點擊右上角的 "+" -> "New repository"。
   3. 為您的倉庫取一個名字 (例如 gemini-aics-chatbot)。
   4. 重要： 請不要勾選 "Add a README file", "Add .gitignore", 或 "Choose a
      license"。因為我們將從本機上傳這些檔案。
   5. 點擊 "Create repository"。

  ---

  第二步：在本機執行 Git 指令


  建立倉庫後，GitHub 會顯示一個標題為 "…or push an existing repository from the command line"
  的區塊。您會需要用到那裡的倉庫網址 (URL)。

  請在您的專案資料夾 (DEV_GeminAICS) 中，依序執行以下指令：

   1. 初始化 Git 倉庫

   1     git init -b main



   2. 將所有檔案加入追蹤 (會自動忽略 .gitignore 中指定的檔案)

   1     git add .


   3. 建立第一個提交 (Commit)

   1     git commit -m "Initial commit: AI chatbot project setup"



   4. 連接到您在 GitHub 上建立的遠端倉庫
      (請將 <YOUR_REPOSITORY_URL> 替換成您在 GitHub 頁面上複製的 .git 結尾的網址)


   1     git remote add origin <YOUR_REPOSITORY_URL>


   5. 上傳您的程式碼

   1     git push -u origin main
